## 第一步：先搞清楚你是哪种情况

搬瓦工登录邮箱找回，大体上分三种情况，处理方式完全不同：

**情况一：知道邮箱，但忘了密码**
这是最好处理的。直接在登录页面点"重置密码"，填入邮箱，收邮件，点链接，改完事。

**情况二：能登录账号，但邮箱填错了或想换邮箱**
进后台直接改。不需要联系客服，自己操作就行。

**情况三：邮箱也不记得了，也登不进去**
这种最麻烦，但也不是没有办法。需要联系官方客服，配合付款记录来核实身份。

## 情况一：只是忘了密码，邮箱还在

这个流程五步走，五分钟之内能搞定。

1. 打开搬瓦工客户端入口，点击右上角的 **Client Area**
2. 在登录框下方找到 **"Forgot Password"**（或 "Request a Password Reset"）
3. 输入你注册时用的邮箱地址，点 **Submit**
4. 去邮箱收一封标题包含 "Password Reset" 的邮件（注意查垃圾箱）
5. 点击邮件里的重置链接，设置新密码，完成

说一下收不到重置邮件的情况——这个坑不少人踩过。先查垃圾邮件箱，那是最高频的原因。如果垃圾箱也没有，检查一下自己填的邮箱地址有没有拼写错误（这就连接到情况三了）。还有一种可能是用的 QQ 邮箱，QQ 邮箱对国外邮件的过滤有时候比较激进，建议注册的时候尽量用 Gmail 或者 Outlook。

## 情况二：账号能登录，但邮箱填错了或者想换

能登进去的话，改邮箱就是两分钟的事：

1. 登录账号后，鼠标移到右上角的 **"Hello, [你的名字]"**
2. 下拉菜单里找到 **"Edit Account Details"**（编辑账户信息）
3. 找到 **Email Address** 这一栏，直接修改成正确的邮箱
4. 其他信息（城市、省份之类的）建议也确认一遍，不用完全真实，但不能明显是假的
5. 最后点 **"Save Changes"** 保存

改完之后建议立刻发一条测试邮件给自己，确认新邮箱能正常收到搬瓦工的邮件。

顺便说一句，搬瓦工账户里有个不太显眼的功能叫 **Email History**（邮件历史记录），位置在右上角下拉菜单里。里面记录了这个账号历史上用过的所有邮箱地址，翻到最后一页能看到最原始的注册邮箱。如果你自己都不确定当时用的是哪个邮箱，可以先去这里查一下。

## 情况三：邮箱记不得了，也登不进账号

不夸张，这种情况挺难受的。但如果你当初付过钱，是有机会找回来的。

核心思路是：**用付款记录来证明你是账号所有者，联系搬瓦工官方客服处理。**

由于无法登录账号，就没办法在后台提交工单（工单系统需要登录）。这时候只能发邮件给搬瓦工官方客服。

**邮件里需要说清楚的内容（用英文写，搬瓦工客服是英文的）：**
- 说明情况：注册邮箱忘记了，无法登录账号
- 提供你能想起来的信息：注册时大概的时间、当时填写的姓名或账单地址
- 附上付款记录：支付宝截图、PayPal 的交易记录、或者信用卡账单截图，要能显示金额和时间

证明材料越充分，处理速度越快。如果只是说"我忘记了，帮我找回"但没有任何凭证，客服基本上也没法处理。

这种方式有结果，但响应时间不定，不是即时的。用过的人普遍反馈大概一两天会有回复，有时候碰上工单量大会慢一些。

## 关于搬瓦工邮箱的几个常见误区

**误区一："换个邮箱重新注册一个就行了"**
如果你没有买过 VPS，或者买了但已经到期了、不在乎了，确实可以这样。但如果账号下面还有正在运行的 VPS，重新注册一个新账号并不能让你找回原来的服务器。服务器是绑在那个账号下的，账号找不回来，服务器就访问不了。

**误区二："QQ 邮箱/163 邮箱都可以注册"**
技术上可以，但实际上这两类邮箱收国外邮件的成功率不太稳定，搬瓦工发出的重置密码邮件、续费提醒邮件，有时候直接进垃圾箱或者根本收不到。建议注册时用 Gmail 或者 Outlook，省事很多。

**误区三："我买的是二手账号，也能找回"**
这里说个实情：搬瓦工账号找回是依据**最原始的注册邮箱**进行的。如果你买的二手账号，原卖家当初注册的邮箱才是"根"，拿那个邮箱才能真正找回账号。这也是为什么搬瓦工在二手圈里被叫做"找回工"——原始邮箱只要还在卖家手里，理论上都能找回去。购买二手账号时，一定要先在账号的 Email History 里确认最原始邮箱是否已经换成了你自己的。

## 常见问题

**Q：搬瓦工的登录邮箱可以改成国内邮箱吗？**
A：可以改，但不建议。国内邮箱服务商对国际来信的过滤策略比较不稳定，搬瓦工发出的账单提醒、密码重置邮件、服务通知，有时候直接被吞掉。换个方向想：如果某天又忘了密码，国内邮箱收不到重置链接，就又回到了最麻烦的那种情况。

**Q：我的账号密码忘了，但注册邮箱也找不到了，还有没有其他办法？**
A：唯一可行的路就是联系客服，附上付款凭证。搬瓦工没有类似"短信验证码找回"这类方式，邮箱是唯一的验证渠道，没有邮箱就得靠人工核实。

**Q：发邮件给客服之后多久会有回复？**
A：没有固定时效，客服是排队处理的。一般情况是一到两个工作日，遇上节假日或者工单量大的时段会慢一些。发完之后耐心等，不用反复发催促邮件。

**Q：登录成功了，但想修改成另一个邮箱，需要验证原邮箱吗？**
A：修改本身不需要验证原邮箱，直接在 Edit Account Details 里改掉 Save 就行。但改完之后原邮箱就失效了，后续重置密码的链接都只会发到新邮箱，注意别再把新邮箱写错了。

**Q：注册搬瓦工用什么邮箱比较好？**
A：Gmail 是最稳的选择，收件率高，国际邮件基本不会有问题。Outlook 也可以。实在用不了这两个，用邮件过滤相对宽松的企业邮箱也行。QQ 邮箱、163 邮箱能用，但碰到收不到邮件的概率比 Gmail 高不少。
